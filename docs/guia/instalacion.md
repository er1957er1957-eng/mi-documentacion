# Guía de Instalación

## Introducción

En esta guía aprenderás cómo instalar correctamente el proyecto en tu ordenador para comenzar a trabajar con él.

## Requisitos Previos

Antes de empezar necesitas tener instalado:

- Git
- Node.js
- Visual Studio Code
- Conexión a Internet

Puedes comprobar si Git está instalado usando:

```bash
git --version
```

## Clonar el Repositorio

Abre una terminal y ejecuta:

```bash
git clone https://github.com/usuario/proyecto.git
```

## Acceder al Proyecto

```bash
cd proyecto
```

## Instalar Dependencias

Para descargar todas las librerías necesarias:

```bash
npm install
```

## Verificar la Instalación

Inicia el proyecto con:

```bash
npm run dev
```

Si todo ha ido bien, aparecerá un mensaje indicando que el servidor está funcionando.
